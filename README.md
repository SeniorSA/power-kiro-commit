# Kiro Power — Commit Standards

Kiro Power para padronização de commits usando Conventional Commits, Semantic Release, Commitlint e Gitmoji.

## Instalação

No Kiro:
1. Abra o painel de Powers (ícone na sidebar)
2. Clique em "Add Custom Power"
3. Selecione "Git Repository"
4. Cole a URL: `https://github.com/SeniorSA/power-kiro-commit`
5. Clique em "Add"

## O que faz

Ensina o Kiro a formatar mensagens de commit seguindo o padrão da equipe:

- Formato: `:emoji: type: subject`
- Emojis padronizados (Gitmoji)
- Tipos do Conventional Commits (feat, fix, docs, etc.)
- Subject em português brasileiro
- Versionamento automático via Semantic Release
